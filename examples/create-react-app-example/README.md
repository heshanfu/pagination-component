## Installation    
* Clone this repo
```sh
git clone https://github.com/dibenso/pagination-component.git
```     
cd to this example
```
cd examples/create-react-app-example
```
* Install dependencies
```sh
yarn install
```     
or
```
npm install
```

## Demo     
To run the demo:      
```sh
npm start
```
    
Then check [http://localhost:3000]()
